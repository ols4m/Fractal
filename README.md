# Fractal
Fractal is an AI search-like tool built for idea synthesis, not just retrieval. It analyzes a query character-by-character to map relationships, generate new connected insights, and presents them as an expandable, easy-to-navigate visual network.
Fractal v1 is a visual research field map, not a search engine or Wikipedia visualizer. Its purpose is to make users say, “I didn’t think about it like that,” by showing the structural forces, tensions, and relationships shaping a big question rather than listing topics or definitions. Each map should contain a small number of clear, approachable core domains, meaningful overlaps that represent real tradeoffs or constraints (not just “related to” links), and one synthesized grey node that captures an emergent pattern or reframing of the entire system. The interface should feel clean and instantly understandable, but clicking deeper should reveal analytical insight about why each force matters and how it interacts with others. It should avoid buzzwords, generic summaries, and random AI-generated fluff, and instead prioritize intentional structure, intellectual clarity, and curiosity-driven exploration over technical complexity or automation.
